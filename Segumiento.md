# MVP: Anotalo

#### Obtjetivo: Registro de Movimientos de un comercio.

#### Objetivos Especificos :

- Gestionar Nomina de Terceros (Clientes | Proveedores)
- Gestionar nomina de Items a comecializar (Productos y Servicios)
- Registro de Movimientos
  - Detalle para seguimiento de Items
